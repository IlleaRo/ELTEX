# Задание 3.10 (POSIX)
Написать две программы, использующиеочередь сообщений: первая отправляет строки в очередь, а вторая – считывает и выводит на экран. Отправляемые строки можно читать из файла или генерировать в программе (например, случайные числа). Для завершения обмена данными первая программа должна отправить сообщение с типом 255.
